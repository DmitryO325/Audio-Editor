#Руководство по программе "Аудиоредактор от DmitryO325":

##Руководство по программе "Проигрыватель":

###Кнопки управления аудиозаписями:

Кнопка "**Воспроизвести**" воспроизводит или запускает аудиофайл, находящийся в плейлисте; если в плейлисте аудизаписей нет, позволяет выбрать пользователю файлы для открытия;

Кнопка "**Пауза**" приостанавливает аудиофайл;

Кнопка "**Стоп**" останавливает работу аудиопотока;

Кнопки "**Перемотка вперёд**" и "**Перемотка назад**" позволяют перемотать аудиофайл вперёд или назад на 10 секунд;

Кнопка "**Повторить композицию**" запускает аудиофайл сначала;

Кнопка "**Предыдущая композиция**" включает предыдущий аудиофайл из плейлиста; если проигрываемый файл являлся первым, то он начинает вопроизводиться сначала;

Кнопка "**Следующая композиция**" включает следующий аудиофайл из плейлиста; если проигрываемый файл являлся последним, то аудиопоток завершается.


###Громкость:

Слайдер громкости позволяет регулировать громкость аудиофайла.


###Кнопки управления плейлистами:

Кнопка "**Открыть плейлист**" позволяет пользователю открыть базу данных с сохранёнными ссылками на аудиофайлы; если в рабочем плейлисте были аудиофайлы, программа спрашивает у пользователя, нужно ли сохранять плейлист;

Кнопка "**Сохранить плейлист**" позволяет пользователю сохранить рабочий плейлист в новой или уже существующей базе данных;

Кнопка "**Добавить в плейлист**" позволяет пользователю добавить файлы в рабочий плйелист;

Кнопка "**Удалить из плейлиста**" удаляет выбранные пользователем аудиофайлы из рабочего плейлиста;

Кнопка "**Очистить плейлист**" очищает рабочий плейлист.


###Кнопки меню:

####Файл:

Кнопка "**Открыть файлы**" позволяет открыть файлы для прослушивания; если в рабочем плейлисте были аудиофайлы, программа спрашивает у пользователя, нужно ли сохранять плейлист;

Кнопка "**Открыть плейлист**" позволяет открыть папку с аудиофайлами для прослушивания, если в рабочем плейлисте были аудиофайлы, программа спрашивает у пользователя, нужно ли сохранять плейлист.


####Плейлист:

Все кнопки данного меню аналогичны кнопкам управления плейлистами.


####Воспроизведение:

Меню "**Скорость**" позволяет выбрать скорость воспроизведения аудиофайла;

Остальные кнопки меню аналогичны кнопкам управления аудиозаписями.


####Аудио:

Кнопки "**Увеличить громкость**" и "**Уменьшить громкость**" увеличивают или уменьшают громкость на 10 пунктов;

Кнопка "**Выключить звук**" отключает звук проигрываемой аудиозаписи.


####Настройки:

Кнопка "**Язык**" открывает окошко с возможностью выбора языка (русский или английский).


####Редактор:

Кнопка "**Открыть редактор**" открывает окно Редактора.


###Работа с плейлистом:

Рабочий плейлист — плейлист, который в данный момент отображается на экране. Он обращается к определённой базе данных, которая хранит ссылки на аудиофайлы.


**Двойной щелчок по строке:**

Двойной щелчок запускает указанный аудиофайл из плейлиста.


**Одиночный щелчок по строке:**

Одиночный щелчок выделяет указанный(е) аудиофайл(ы).


###Работа со слайдером:

Слайдер позволяет изменить время аудиозаписи, если *перетаскивать* слайдер. Если просто нажать на какое-то место, ничего не произойдёт (будет исправлено).


###Горячие клавиши:

F — Воспроизвести;\
G — Пауза;\
H — Стоп;\
J — Предыдущая композиция;\
K — Повторить композицию;\
L — Следующая композиция;\
N — Перемотка назад на 10 секунд;\
M — Перемотка вперёд на 10 секунд;\
A — Открыть плейлист;\
S — Сохранить плейлист;\
D — Добавить в плейлист;\
C — Выключить звук;\
V — Уменьшить громкость на 10 пунктов;\
B — Увеличить громкость на 10 пунктов;\
E — Открыть Редактор.


##Руководство по программе "Редактор":

###Работа с данными аудиофайла:

Чтобы изменить данные планируемого аудиофайла, нужно щёлкнуть по таблице двойным кликом то место, которое хотите изменить.


###Работа с аудио слайдером:

Работает также перетаскиванием. Используется для обрезки аудиофайла.


###Время аудиозаписи:

Текущее и максимальное время исходной аудиозаписи.


###Итоговое время аудиозаписи:

Промежуток времени, который будет в итоговой аудиозаписи.


###Кнопки обрезки:

Кнопки "**Обрезать слева**" и "**Обрезать справа**" задают крайний левый и крайний правый промежуток соответственно.


###Работа со слайдером громкости:

Задаёт громкость в процентах относительно исходной аудиозаписи.


###Путь:

Показывает путь, по которому находится исходный аудиофайл.


###Новый путь:

Путь, по которому планируется открыть новый файл для работы.


###Кнопки открытия:

Кнопка "**Открыть файл по ссылке**" открывает аудиофайл, который указан в "Новом пути";

Кнопка "**Выбрать файл**" позволяет открыть файл для работы.


###Кнопка "**Сохранить аудиофайл**:"

Сохраняет итоговую версию файла в указанном пользователем месте.


###Горячие клавиши:

A — Выбрать файл;\
S — Сохранить файл;\
K — Обрезать слева;\
L — Обрезать справа.
